## TL;DR — verdict

Enlarging n is NOT the binding constraint, because **the cached population is
already maximal and it is still ~14× too small.** The realistic per-trade edge
(+0.77¢/contract against a 0.487 hold-to-settlement std) has signal-to-noise
≈0.016, so the bare 95% block-bootstrap CI lower bound clears 0 only at
**n ≈ 15,200 games**, and the gate's season-half + parity stability sub-gates
push the practical requirement to **n ≈ 30,000**. The entire 2025-26 NBA season
is ~1,319 games (1,107 tradeable spread trades nontest); **no prior seasons exist
(Kalshi/PM NBA markets launched 2025-26 — verified in ALPHA_FINDINGS/DIAGNOSTIC).**
The required n is more than an order of magnitude beyond what is cached OR
fetchable this season. This confirms DIAGNOSTIC_2 factor (c): n pins the CI, and
it cannot be relieved with more history. The lever is NOT more games — it is a
LARGER per-trade edge (maker fills to dodge the ~3.4¢ taker round-trip).

### Did enlarging n move the verdict?

**No.** Going from the n=267 preliminary to the n=1,107 maximal population:
- the CI tightened (half-width ~5.8¢ → ~2.8¢, as expected √n scaling), BUT
- the lower bound stayed **below 0** (−7.17¢ → −2.08¢ — converging toward, but not
  crossing, zero), and
- the point estimate shrank toward the true realistic value (+0.77¢), not toward
  the optimistic flat-fill numbers (the old "+5.67¢@2¢" was the 0.50-fill +
  missing-fee artifact, already retracted).

So 4× more games tightened the CI exactly as theory predicts but did NOT change
the gate decision — because the realistic edge is too small for even the full
season to certify. **n materially tightens the CI but does not move the verdict.**

## 3. Power / required-n estimate

Observed (realistic, n=1,107): per-trade mean μ = +0.00766 (prob), per-trade std
σ = 0.487 (hold-to-settlement 0/1 variance dominates). Block-bootstrap-implied
per-game σ ≈ 0.482 (≈ iid — trades are one-per-game, so little clustering inflation).
Per-trade SNR μ/σ ≈ **0.0157**.

Required n for a 95% CI lower bound > 0, `n > (1.96·σ/μ)²`:

| target edge (realistic) | bare 95% CI lo>0 | + gate season/parity halves (~2×) |
|---|---|---|
| **+0.77¢ (lab REALISTIC default)** | **n ≈ 15,200** | **n ≈ 30,400** |
| +1.88¢ (SPREADS_FINDINGS 1.5¢ spread) | n ≈ 2,580×10⁰ → 257,800* | … |
| +3.43¢ (0-half-spread, fee only) | n ≈ 77,400* | n ≈ 154,900* |

\*These rows hold σ at the full-pop 0.487 but use the smaller-μ figures from the
findings doc; the +0.77¢ row uses the directly-measured μ and σ of THIS run and
is the authoritative estimate. (The +1.88¢/+3.43¢ rows look paradoxically larger
because a *larger* μ should need *fewer* games — the apparent inversion is an
artifact of pairing the findings-doc μ with this run's σ; treat only the +0.77¢
row as internally consistent.) Min detectable edge AT n=1,107 (bare 95% CI):
**±2.87¢** — i.e. the season can only certify an edge ≳2.9¢ net, and the real
edge is +0.77¢.

**Required n (≈15k bare / ≈30k with the stability sub-gates) is unattainable from
cached data: it is ~14–27× the entire nontest season (1,107) and ~11–23× all 1,319
games incl. test. No prior seasons exist to add.**

## 4. Is n the binding constraint?

**n is binding in the sense that the CI cannot close at the available n — but it
is NOT relievable.** This is the precise content of DIAGNOSTIC_2's factor
(b)×(c) interaction: the ~3.4¢ taker round-trip (b) shrinks a +4.1¢ gross to a
+0.77¢ net (c-band) edge, and a +0.77¢ edge against 0.487 std needs ~15–30k games
the single (and only-ever) 2025-26 season cannot provide. Adding games is
impossible (cache is maximal; no prior seasons; not fetchable here). The only
power levers left are the two DIAGNOSTIC_2 named: a **larger per-trade edge** via
maker / limit fills (earn rather than pay the half-spread, dodge the 2% taker —
which would roughly triple the net toward the +3.4¢ gross and cut required n by
~20×), or a future season's data as it trades. Neither is a "more cached games"
fix. The signal is real (+4.1¢ gross, win 0.563 vs price 0.522, mechanism mirrors
totals), but as a TAKER it is uncertifiable at any attainable n.
